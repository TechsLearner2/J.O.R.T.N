## TUTORIAL FOR INSTALLATION
### -FOR WINDOWS-  
Hello Player,
Checkout at Our Release
1. Install the Zip File by clicking it
2. Extract the file to desired place
3. Open the game application
4. Enjoy!


### -FOR LINUX-  
Hello Player,

5. Enjoy!
